### User Stories ###

# As a visitor

As a visitor, I want to be able to register 
so I can book trips 

As a visitor, I want to be able to specifiy travel country 
so I can visit a country of choice in my travel 

As a visitor, I want to be able to specifiy travel region 
so I can visit a region of choice in my travel 

As a visitor, I want to be able to specifiy travel city 
so I can visit a city of choice in my travel 

As a visitor, I want to be able to set criterias for the hotels presented to me 
so I can book a hotel that meets my requirements 

As a visitor, I want to be able to set max total price for my trip 
so I can ensure I view trips that are within my budget 

As a visitor, I want to be able to see the cost of my suggested trips 
so I can be sure of how much I will have to pay 

As a user, I want to be able to specify details for my co Travelers in a booking 
so I can book the trip everyone Im traveling with 

As a visitor, I want to be able to set travel dates for my trip 
so I can choose a trip that matches my schedule 



# As a user

As a user, I want to be able to book a trip when I've found a trip that matches my requirements 
to be able to go on my vacation. 

As a user, I want to be notified to that my trip has been booked 
so I can ensure my trip has been accepted 

As a user, I want to be able to view my previous bookings 
so I can get an overview of previous trips 

As a registered user i want o be able to finns a information about hotel position and see distance to important places that i would like to visit. 

As a registered user, I want to filter accommodations by simple preferences (WiFi, breakfast included, parking, pet-friendly) so that I can quickly find places that meet my basic needs. 

As a registered user i want be able to reserv hotels ith extra rooms to my grupp stay together during the trip. 

As a registered user, I want to see when a hotel is fully booked and get suggestions for similar hotels nearby so that I can still find a place to stay 



# As a employee

As one employee, I want to be able to search bookings with different criterias to be able to help customer fast in order to solve their problem.

As one employee, I want to have possibilitiy to open and read support tickets to be able to understand what customer need in order to give correct assistance.

As one employee, I want to be able to see users, bookings, hotels and travel packages in list and details to be able to manage the system in order to keep data updated.

As one employee, I want to be able to add new activity/theme for trips to be able to create more travel experiences in order to attract more customers.

As a support personel, I want to be able to view bookings based on search critera so I can provide assistance to a customer that Contacts us 


# As a admin

As an admin, I want to see all users, bookings, hotels, and travel packages in lists and details so that I can manage everything in the system. 

As an admin, I want to change room availability and mark hotels as fully booked so that customers only see correct information.

As an admin, with sufficent privileges, I want to be able to add Another admin so I can add a new employee 

As an admin, with sufficent privileges, I want to be able to add a new hotell so we can provide bookings for new hotels 

As an admin, with sufficent privileges, I want to be able to add a new activity for trips so we can provice new themes for trips 

As an admin, with sufficent privileges, I want to be able to blacklist a user so we can prevent troublesome clients to cause further issues 

As an admin, with sufficent privileges, I want to be able to view a log of user activity so we can determine what a user did at a specific time or event 

As an admin, with sufficent privileges, I want to be able to view a log of site traffic so we can determine user behaviour and origin 

As an admin, with sufficent privileges, I want to be able to remove a travel location in case we're not able to/dont want to book travels to that location
